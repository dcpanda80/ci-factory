Getting Started

[This is preliminary documentation and subject to change.]

This topic will show you NAnt building itself.  Before you begin make sure your system meets the system requirements.

  1. Download the latest release.
  2. Unzip the distribution into a new folder (e.g. C:\Tools\CI Factory).
  3. Open a command prompt and change the directory to the folder where NAnt was installed (e.g. C:\Tools\CI Factory\nAnt\bin).
  4. Type nant -h to see the commandline help.
